# Lab 8: Android Tutorial

In this lab you will be given code for a functional tip calculator app.

The most important files are:

* [`MainActivity.java`](./app/src/main/java/org/academicode/tipcalculator/MainActivity.java)
* [`Result.java`](./app/src/main/java/org/academicode/tipcalculator/Result.java)
* [`main.xml`](./app/src/main/res/layout/main.xml)
* [`result.xml`](./app/src/main/res/layout/result.xml)
* [`strings.xml`](./app/src/main/res/values/strings.xml)

## Problem

To pass, you must add a few features and correct a handful of problems:

* Correct the spelling errors (Hint: look at the `strings.xml` file).
* Add a new 5% button to the `.xml` document and add a click listener for it.
* Reformat the `.xml` files to accommodate the new 5% button above the 10% button.
